# 懒人实词检索
可以使你省去目测实词的痛苦而无用的时光  
##使用方法
在文件夹中创建[^“text.txt”]，把目标文本复制~~或打入~~到文件中，运行“懒人诗词检索.exe”，稍等片刻
##可能问题
+杀毒软件报毒
	允许运行即可
+输出乱码
	打开“text.txt”，查看窗口右下角，若显示“ANSI”或“GB2312-80”，则另存为——文件编码设为“UTF-8”，重新打开exe
