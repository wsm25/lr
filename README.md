# 懒人实词检索
可以使你省去目测实词的痛苦而无用的时光  

## 使用方法
在文件夹中创建[^“text.txt”]，把目标文本复制~~或打入~~到文件中，运行“懒人诗词检索.exe”，稍等片刻  
[^“text.txt”] : 文件名不可改变！  

## 文件说明

| 文件 | 作用 |
|  ----  | ----  |
| 懒人诗词检索.exe | 主文件，不可删除 |
| 1.vbs | 帮助文件，最好不要删除 |
| lr.html | 运行软件后文本结果，可打开，可删除 |

## 可能问题
+ 杀毒软件报毒  

	允许运行即可
+ 输出乱码  

	打开“text.txt”，查看窗口右下角，若显示“ANSI”或“GB2312-80”，则另存为——文件编码设为“UTF-8”，重新打开exe
